MatrixBasedConjugateGradients¶

class probnum.linear_solvers.MatrixBasedConjugateGradients[source]¶

Bases: probnum.linear_solvers.ProbabilisticLinearSolver

Conjugate Gradients using prior information on the matrix inverse.

In the setting where \(A\) is a symmetric positive-definite matrix, this solver takes prior information either on the matrix inverse \(H=A^{-1}\) and outputs a posterior belief over \(H\). This code implements the method described in 1.

1

Hennig, P., Probabilistic Interpretation of Linear Solvers, SIAM Journal on Optimization, 2015, 25, 234-260
